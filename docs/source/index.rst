
Welcome to SoloTE documentation!
===================================

**SoloTE** is a Python tool to calculate the expression of Transposable Elements (TEs) in 10X Genomics data, such as **Chromium** single-cell, and **Visium** spatial transcriptomics.

It is intended to work **directly** on the BAM files generated by CellRanger or SpaceRanger, or an alignment performed with `STARsolo <https://github.com/alexdobin/STAR/blob/master/docs/STARsolo.md>`_

First, check out the :doc:`installation` for information on how to prepare your computer, and then the :doc:`inputfiles`. Once you are ready, check :doc:`running_solote` for information on how to start running SoloTE.


For technical details on SoloTE, please read its publication:

.. Important::

   | *SoloTE for improved analysis of transposable elements in single-cell RNA-Seq data using locus-specific expression*
   | Communications Biology volume 5, Article number: 1063 (2022)
   | https://www.nature.com/articles/s42003-022-04020-5
   | https://doi.org/10.1038/s42003-022-04020-5
